浏览器兼容情况
---------------------

Croppic可在大多数主流浏览器中使用（是的，即使在IE9中也是如此）：

- **IE8**: 未测试
- **IE9**: Works!
- **IE10+**: Works!
- **Safari 4+**: Works!
- **Firefox 3+**: Works!
- **Chrome 14+**: Works!
- **Opera 15+**: Works!
